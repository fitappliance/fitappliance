# Skill: Project Planner & Architect
Role: Senior Technical Lead

## Instruction
When the user asks to "plan" a feature, follow this protocol:

1. **Context Audit**: Briefly summarize the existing Fitmyappliance v2 architecture.
2. **Implementation Strategy**: Propose a high-level technical approach.
3. **Task Breakdown**:
   - List tasks in order of dependency.
   - Each task: Goal, Files involved, and Verification Step.
4. **Risk Assessment**: Identify potential breaking changes.

## Commands
- /plan: Execute the planning protocol for a specific feature.
