# PDF pipeline fixture

`sample-bosch-fridge.pdf` is a compact text-based fixture derived from Bosch's public manufacturer specification sheet for the B36FD52SNS refrigerator:

https://media3.bosch-home.com/Documents/25590041_B36FD52SNS_Bosch_Pre_Spec_Sheet_Refrigeration_Freestanding.pdf

The original manufacturer PDF is larger than the repository fixture budget. This fixture keeps the relevant factual installation text and dimensions in a tiny PDF so pipeline tests can exercise real PDF text extraction without committing large binary files.
